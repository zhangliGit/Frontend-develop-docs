# javascript核心概念

## javascript引擎

+ javascript引擎用来编译和运行javascript代码

+ javascript引擎先把源代码编译成ast（抽象语法书），然后在通过jit(即时编译)转化为本地机器码执行，此过程中会对语法进行优化

+ javascript是动态编译，只有在函数调用时才会编译成本地代码（机器码），这样就提高了响应时间减少了时间开销
